# 红宝书词典 🔍

纯前端版本：无需后端服务器，在浏览器中即可搜索单词并查询其在红宝书中的页码。

## 功能

- **精确匹配**：输入完整单词，快速定位
- **前缀匹配**：输入单词前缀，展示所有以该前缀开头的单词
- **包含匹配**：输入片段，找到包含该片段的所有单词
- **拼写纠错**：基于 Levenshtein 距离的拼写相近提示

## 本地使用

直接在浏览器中打开 `index.html` 即可使用。无需任何安装或配置。
